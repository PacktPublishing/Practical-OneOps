Docker Engine Cookbook
======================
A cookbook to install Docker. Docker engine is a portable, lightweight application runtime and packaging tool.

Requirements
------------
* OS     : **>= CentOS/RHEL 7.x**
* Kernel : **>= 3.10.x**
* Ruby   : **>= 2.x**

Usage
-----
