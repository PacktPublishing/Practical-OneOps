Etcd Cookbook
=============
A cookbook to install and configure etcd. Etcd is a distributed, consistent key-value store for shared configuration and service discovery.


Requirements
------------
* OS     : **>= CentOS/RHEL 7.x**
* Kernel : **>= 3.10.x**
* Ruby   : **>= 2.x**


Usage
-----
#### etcd::add
